# diabetes-detector
### A deep learning model to detect diabetes from the information you provide.

# You need to input these fields for detection:
### 1. Pregnancies, 
### 2. Glucose,
### 3. Blood Pressure,
### 4. SkinThickness, 
### 5. Insulin,
### 6. BMI, 
### 7. DiabetesPedigreeFunction,
### 8. Age.
